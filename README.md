# ENux-AI

ENux AI, is an AI built for general purpose and ENux, that uses the 7B Qwen 2.5 Model, running locally.
This AI is suitable for daily talking, coding and talking about ENux

## Before Getting Started

- System requirements:
  - Minimum: 8 GB of RAM, a 4 Core CPU
  - Recommended: 12 GB of RAM, a 6 Core CPU
 
- OS:
  - Must be either Debian, or ENux for max compatability, and less headaches
 
## How to Install

- Install nodejs by running `apt install nodejs`
- Run `sudo make`
- To use ENux AI, run `enux-ai`
- **Enjoy**
